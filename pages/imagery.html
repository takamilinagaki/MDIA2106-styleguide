Graphics and Icons

I wanted all the buttons to hover into the same color because it will make it easier for user to know what is clickable

Green for submit Buttons because it is easily noticable in a page

Blue for text container button when there is no submit Button. Users will learn that the blue buttons are clickable because they will have hover.

Small Blue Buttons will be used for Category page so users can click on their option that they have chosen

Medium Blue Buttons will be used for Reference Page and Healthy Habits Page, it has the same box size as the info boxes

Bigger Blue Buttons will be used for the Identification page to make it easier to click

Hamburger Menu Icons will be the same for all the pages because it is recognizable as a drop down menu
